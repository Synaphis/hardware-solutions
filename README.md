# Aorixis - Enterprise IT Hardware & Consultation

Aorixis is a high-credibility, premium B2B platform for enterprise IT infrastructure procurement. We specialize in supplying certified Tier-1 hardware, including servers, storage arrays, and networking equipment, backed by professional engineering expertise.

## Core Services

- **Direct Hardware Provisioning:** Extensive local inventory of rigorously tested, enterprise-grade hardware.
- **Custom Configuration:** OEM-certified engineers build systems to exact technical specifications.
- **Technical Consultation:** Expert guidance from data center architects and virtualization specialists.
- **ITAD & Asset Liquidation:** Secure and fair market valuations for decommissioned enterprise assets.

## Tech Stack

- **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Email:** Nodemailer with Zoho Mail integration

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Set up environment variables:**
    Create a `.env.local` file based on `.env.example` with your Zoho SMTP credentials.

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
